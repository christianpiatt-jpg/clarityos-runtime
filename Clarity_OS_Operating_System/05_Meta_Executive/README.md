# Meta Executive Layer 
The Meta Executive Layer oversees how decisions are made, not just what decisions are made. 
It monitors drift, distortion, overclocking, and collapse in the Executive Layer. 
 
## Purpose 
- Ensure decisions come from clarity, not pressure. 
- Detect when the operator is acting outside their frame. 
- Maintain decision integrity across time. 
- Provide reflective oversight for the entire OS. 
 
## Relationship to the OS 
- Geometry defines the frames the Meta Executive evaluates. 
- ULDA provides the loop the Meta Executive checks for integrity. 
- Martial OS stabilizes the operator so reflection is possible. 
- Executive Layer provides the decisions being evaluated. 
 
## Files in this subsystem 
- LOGIC.md: the logic of meta-level evaluation. 
- PRACTICES.md: reflective practices and review cycles.
