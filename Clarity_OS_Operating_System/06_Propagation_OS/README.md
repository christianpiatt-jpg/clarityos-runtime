# Propagation OS 
Propagation OS describes how ideas, behaviors, signals, and structures spread through environments. 
It models influence, drift, contagion, stabilization, and systemic change. 
 
## Purpose 
- Predict how patterns will move through a system. 
- Identify where clarity strengthens or weakens. 
- Understand how pressure or distortion spreads. 
- Support ethical, stable propagation of healthy patterns. 
 
## Relationship to the OS 
- Geometry defines the channels and fields propagation moves through. 
- ULDA diagnoses the pattern being propagated. 
- Martial OS stabilizes propagation under pressure. 
- Executive Layer issues actions that initiate or redirect propagation. 
- Meta Executive evaluates whether propagation is healthy or distorted. 
 
## Files in this subsystem 
- ENGINE.md: the mechanics of propagation. 
- PRACTICES.md: patterns and drills for ethical propagation.
