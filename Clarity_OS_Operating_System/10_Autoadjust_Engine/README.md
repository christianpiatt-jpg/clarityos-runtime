# Autoadjust Engine 
The Autoadjust Engine monitors operator state, altitude, drift, and pressure, and recommends corrections automatically. 
It does not override operator agency; it provides real-time adjustments to maintain clarity and stability. 
 
## Purpose 
- Detect drift, pressure, collapse, or overclocking. 
- Suggest altitude shifts (v2, founder, global, planetary). 
- Activate the correct Buddy automatically. 
- Re-run ULDA phases when skipped. 
- Maintain operator clarity without taking control. 
 
## Files 
- RULES.md: detection rules. 
- FLOW.md: autoadjust sequence. 
- MODES.md: autoadjust modes (soft, moderate, aggressive).
