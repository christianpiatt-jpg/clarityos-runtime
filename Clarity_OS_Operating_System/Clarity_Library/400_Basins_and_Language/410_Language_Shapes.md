\# Language Shapes

\## Dewey: 410



\## Summary

Languages have shapes—linear, circular, contextual, hierarchical—that determine how meaning is encoded, transmitted, and distorted.



\## Core Concepts

\- Linear languages

\- Contextual languages

\- Hierarchical languages



\## Body

Language shape determines how meaning flows. Linear languages prioritize sequence. Contextual languages prioritize relationship. Hierarchical languages prioritize authority. Operators identify language shape to predict how meaning will be interpreted, distorted, or resisted in a given basin.
