# 🟦 **CONSOLE OS v1.1 — Mode‑Dependent Pane Reconfiguration**

### **The v1.1 Principle**
In v1.0, all four panes were always visible.  
In **v1.1**, the console **reconfigures itself** depending on the operator mode.

Each mode:

- **elevates one pane**  
- **shrinks two panes**  
- **ghosts one pane**  
- **re-centers the command spine**  

This is exactly how real operator consoles behave under load.
