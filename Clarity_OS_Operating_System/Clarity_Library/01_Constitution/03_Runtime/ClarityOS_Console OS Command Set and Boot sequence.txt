# 🖥️ **CONSOLE OS — COMMAND SET v1.0**  
*The operator‑grade command language for navigating the ClarityOS Console.*

Commands are grouped by function, not layer.

---

# 🔹 **I. SYSTEM COMMANDS**
These control the console itself.

### **`console.open`**  
Launches Console OS into Observe Mode.

### **`console.mode <observe|stabilize|navigate|repair|govern>`**  
Switches the console’s operating mode.

### **`console.reset`**  
Returns all panes to zero‑state.

### **`console.freeze`**  
Locks the current state for analysis.

### **`console.unfreeze`**  
Releases the lock and resumes live updates.

---

# 🔹 **II. RETRIEVAL COMMANDS**
These pull artifacts from the library.

### **`get.tag <tag>`**  
Retrieve an artifact by tag.

### **`get.dewey <code>`**  
Retrieve an artifact by Dewey code.

### **`get.module <module>`**  
Retrieve an artifact by module name.

### **`get.event <event-type>`**  
Retrieve the ELINS event profile.

### **`get.basin <basin>`**  
Retrieve basin‑specific variants.
