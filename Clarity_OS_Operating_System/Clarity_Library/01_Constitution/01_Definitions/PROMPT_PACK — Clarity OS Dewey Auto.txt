CANON STATUS CHECK

Determine whether this text is:
- Canonical

Rules:
Canonical if it:
- Defines invariants
- Defines diagnostic engines
- Defines subsystem architecture
- Uses finalized specification language

Experimental if it:
- Is a draft, patch, update, or working note
- Explores hypotheses or alternatives
- Contains operator or scratchpad language

Primary Dewey Code Selection:

- 600 — Interpretation rules / canon control

Conflict Rule:

that governs the document’s primary purpose.
