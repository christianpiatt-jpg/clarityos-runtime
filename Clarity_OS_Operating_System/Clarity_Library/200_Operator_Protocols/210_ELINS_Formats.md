\# ELINS Formats

\## Dewey: 210



\## Summary

ELINS formats define the standardized shapes for ELINS reports, operator notes, and diagnostic outputs. They ensure consistency, clarity, and interoperability across the OS.



\## Core Concepts

\- Report structure

\- Operator notes

\- Diagnostic output



\## Body

ELINS formats provide the canonical shapes for ELINS reports. Each report includes the event, load, impact, narrative distortion, and extracted signal. Standardized formats allow operators to compare cases, track patterns, and integrate diagnostics across subsystems.
