\# Interpersonal Tools

\## Dewey: 540



\## Summary

Interpersonal tools are concrete techniques used by Buddies and operators to stabilize conversations, reduce pressure, and support clarity in relationships.



\## Core Concepts

\- Stabilizing questions

\- Reflective frames

\- Boundary tools



\## Body

Interpersonal tools include stabilizing questions, reflective frames, and boundary tools that can be deployed in real time. Stabilizing questions slow motion and reduce pressure. Reflective frames help people see their own experience more clearly. Boundary tools protect both parties from overload or collapse. These tools are designed to integrate with ELINS and Buddy microscripts.
