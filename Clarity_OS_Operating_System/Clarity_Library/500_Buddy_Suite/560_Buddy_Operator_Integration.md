\# Buddy–Operator Integration

\## Dewey: 560



\## Summary

Buddy–Operator integration defines how Buddy tools and operator protocols work together. It ensures that interpersonal support and system diagnostics stay aligned.



\## Core Concepts

\- Role boundaries

\- Escalation paths

\- Diagnostic handoff



\## Body

Buddy–Operator integration clarifies when a situation stays in the Buddy layer and when it escalates to operator protocols. It defines role boundaries, escalation paths, and diagnostic handoffs. Buddies focus on interpersonal stabilization; operators handle system-level diagnostics and interventions. Integration prevents overload on either side and keeps the whole system coherent.
