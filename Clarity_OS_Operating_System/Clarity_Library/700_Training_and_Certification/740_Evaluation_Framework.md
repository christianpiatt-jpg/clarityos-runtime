\# Evaluation Framework

\## Dewey: 740



\## Summary

The evaluation framework provides structured methods for assessing operator performance, stability, and clarity across drills, simulations, and real-world scenarios.



\## Core Concepts

\- Assessment criteria

\- Performance metrics

\- Stability indicators



\## Body

The evaluation framework defines how operator performance is measured. It includes clarity metrics, stability indicators, diagnostic accuracy, and field-safety adherence. Evaluations are designed to be objective, repeatable, and aligned with the Training Canon and Operator Tiers.
