\# Pressure-Shape Diagnostics

\## Dewey: 330



\## Summary

Pressure-shape diagnostics reveal how pressure interacts with institutional geometry. They identify weak points, load concentrations, and collapse vectors.



\## Core Concepts

\- Load concentration

\- Weak-point detection

\- Collapse vector mapping



\## Body

Pressure interacts with institutional shape in predictable ways. Some shapes amplify pressure; others dissipate it. Pressure-shape diagnostics allow operators to identify where collapse will begin and how it will propagate. This is essential for stabilizing institutions under load.
