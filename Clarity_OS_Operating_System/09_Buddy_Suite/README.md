# Buddy Suite 
The Buddy Suite provides operator-side tools that support clarity, stability, and decision-making. 
Buddies are lightweight modules that run alongside the operator, offering guidance, diagnostics, and structure. 
 
## Purpose 
- Provide real-time support for ULDA loops. 
- Help operators maintain clarity under pressure. 
- Offer micro-tools for reflection, stabilization, and decision quality. 
- Serve as companions for daily operation of the OS. 
 
## Relationship to the OS 
- Geometry defines the space Buddies interpret. 
- ULDA provides the loop Buddies support. 
- Martial OS provides stability under pressure. 
- Executive Layer provides commands Buddies help execute. 
- Meta Executive provides oversight Buddies reinforce. 
- Propagation OS models how Buddy outputs spread through systems. 
- Global OS provides the macro context Buddies adapt to. 
 
## Files in this subsystem 
- OVERVIEW.md: the purpose and structure of the Buddy Suite. 
- MODULES.md: the list of Buddy modules and their functions.
