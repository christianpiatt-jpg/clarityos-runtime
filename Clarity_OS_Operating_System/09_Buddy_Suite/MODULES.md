# Buddy Modules 
Buddy modules are specialized micro-tools that support the operator in specific tasks. 
Each module is designed to be simple, stable, and easy to run in real time. 
 
## Core Buddy Modules 
- Clarity Buddy: helps maintain interpretive clarity and frame integrity. 
- Stability Buddy: monitors pressure and internal geometry. 
- Decision Buddy: assists with ULDA Diagnose and Executive commands. 
- Drift Buddy: detects drift, hijack, or frame collapse. 
- Review Buddy: guides post-action reflection and loop review. 
 
## Buddy lifecycle 
- Initialize: load the Buddy with the current context. 
- Run: provide real-time support during operations. 
- Capture: record key signals or decisions. 
- Review: summarize insights for Meta Executive. 
 
## Why Buddies matter 
Buddies make the OS usable in real time. 
They reduce cognitive load, increase stability, and strengthen clarity. 
They ensure the operator never runs ULDA alone.
