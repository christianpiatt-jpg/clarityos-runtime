# Buddy Suite Overview 
The Buddy Suite is a collection of micro-modules that assist the operator in real time. 
Each Buddy focuses on a specific function: clarity, stability, reflection, or execution. 
 
## Core Buddy functions 
- Stabilization: help the operator maintain internal geometry. 
- Diagnostic: help identify drift, pressure, or pattern mismatch. 
- Reflective: help review decisions and ULDA loops. 
- Executive: help issue and track commands. 
 
## Buddy design principles 
- Lightweight: Buddies must not overload the operator. 
- Modular: each Buddy handles one function well. 
- Inspectable: Buddies must be easy to understand and review. 
- Ethical: Buddies must support agency, not override it. 
 
## Integration with ULDA 
- Buddies support Observe by highlighting signals. 
- Buddies support Locate by mapping signals to geometry. 
- Buddies support Diagnose by identifying patterns. 
- Buddies support Adjust by suggesting interventions. 
- Buddies support Review by summarizing loops.
