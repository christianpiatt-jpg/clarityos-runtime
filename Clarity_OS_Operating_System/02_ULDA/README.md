# ULDA Kernel 
ULDA (Universal Loop Diagnostic Architecture) is the core diagnostic loop of the Clarity OS. 
It describes how an operator or system observes, interprets, decides, and acts in a stable, repeatable way. 
 
ULDA is not a script; it is a loop that can run at many scales: moment-to-moment, session, project, or system. 
It is designed to be inspectable, teachable, and debuggable under pressure. 
 
## Relationship to the rest of the OS 
- Environmental Geometry defines the space ULDA runs inside. 
- Martial OS stabilizes ULDA under conflict and pressure. 
- Executive Layer turns ULDA outputs into concrete commands. 
- Meta Executive watches how ULDA is being used and adjusts patterns. 
- Propagation OS describes how ULDA-shaped decisions spread. 
 
## Files in this subsystem 
- KERNEL.md: defines the core loop, phases, and invariants. 
- FLOW.md: describes how information and decisions move through the loop.
