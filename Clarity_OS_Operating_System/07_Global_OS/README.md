# Global OS 
Global OS models large-scale systems, cross-regional dynamics, and planetary patterns. 
It provides the macro-level context in which all other subsystems operate. 
 
## Purpose 
- Understand large-scale forces shaping environments. 
- Identify global constraints that influence local decisions. 
- Model systemic drift, instability, and long-term trends. 
- Provide a planetary frame for ULDA and Executive decisions. 
 
## Relationship to the OS 
- Geometry defines the spatial layers Global OS operates across. 
- ULDA diagnoses global patterns and cycles. 
- Martial OS stabilizes operators navigating large-scale pressure. 
- Executive Layer issues actions that interact with global constraints. 
- Propagation OS models how global patterns spread across regions. 
 
## Files in this subsystem 
- MODEL.md: the global model and its components. 
- DIAGNOSTICS.md: checks and indicators for global stability.
