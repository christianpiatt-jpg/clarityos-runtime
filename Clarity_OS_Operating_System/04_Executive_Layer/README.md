# Executive Layer 
The Executive Layer converts clarity into concrete action. 
It is the subsystem responsible for issuing commands, sequencing actions, and ensuring follow-through. 
 
## Purpose 
- Turn ULDA diagnoses into decisions. 
- Translate decisions into commands. 
- Maintain action integrity across time. 
- Ensure the operator stays in agency, not reaction. 
 
## Relationship to the OS 
- Geometry defines the environment where actions land. 
- ULDA provides the diagnosis that informs decisions. 
- Martial OS stabilizes the operator under pressure. 
- Meta Executive monitors whether decisions are clean or distorted. 
 
## Files in this subsystem 
- OVERVIEW.md: the structure and purpose of the Executive Layer. 
- COMMANDS.md: the command grammar and action types.
