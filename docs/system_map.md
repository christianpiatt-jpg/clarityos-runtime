# System Map

## Purpose

The System Map specifies the nodes of ClarityOS and the flows between them.

## Nodes

- Hero
- Navigation
- Cockpit
- Vault
- Five Layers
- State Engine
- Geometry System
- ELINS
- Badge
- Runtime

## Flows

- **Cognitive flow** — cyan.
- **Data flow** — muted, dashed.
- **State flow** — red.
- **Geometry flow** — polyhedral.

## Layout

- Navigation at the top.
- ELINS above center.
- The five layers at center.
- Vault to the left.
- Cockpit and State Engine to the right.
- Badge at lower right.
- Runtime at the bottom.

## Mobile Variant

- All nodes render in a single vertical stack.
